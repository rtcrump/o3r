# O3R Lab Website

Single-page Eleventy site for Ocular Oncology Outcomes Research, designed for GitHub Pages.

## Requirements

- Node.js 20+
- npm

## Local development

```bash
npm install
npm run start
```

## Build

```bash
npm run build
```

## Content files

Edit these files to update the rendered site:

- `src/_data/site.json`
- `src/_data/research.json`
- `src/_data/projects.json`

The Team and Publications sections are intentionally not rendered at launch. Their scaffolding can be populated later when the roster and citation strategy are ready.
